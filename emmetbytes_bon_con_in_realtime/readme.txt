EMMETBYTES WOULD LIKE TO THANK YOU FOR PURCHASING THIS MODULE.

please take note : because this module fetched a lot of datas if it is used fully, emmetbytes would like to recommend that your
website should be on a vps or dedicated server before installing this module.

this module is intended to display the default boonex modules contents in real-time. after you or a site member will create an
entry on your website a specific block would be able to display the contents without refreshing your websites pages. it will also
refresh the entry when a certain data of the entry changes.

EVENTS BLOCKS : 
    DESCRIPTION :
    - this would display the events datas or events contents in real-time. the fetching of the datas was based on the way the
        default boonex events blocks gets its datas

    COMMON ADMINISTRATION SETTINGS :
        - on the administration page, the administrator can choose whether to display the datas automatically or to display the new entries button.
            it would also allow the administrator to set for the maximum number of datas to be displayed, to dispaly the author, to
            display the date, to display the location, to display the fans count and to display the rating.

    LIST OF AVAILABLE BLOCKS :
        HOMEPAGE BLOCK :
            - PUBLIC EVENTS IN REALTIME
                - this block also contains the upcoming, featured, recently added, top and popular tabs.
                additional administration settings :
                    - on the administration page, the administrator can also choose the default tab for this block.
        PROFILE BLOCKS :
            - MY EVENTS IN REALTIME
            - JOINED EVENTS IN REALTIME
        MAIN EVENTS PAGE BLOCKS :
            - UPCOMING PUBLIC EVENTS IN REALTIME 
            - PAST PUBLIC EVENTS IN REALTIME
            - RECENTLY ADDED EVENTS IN REALTIME
